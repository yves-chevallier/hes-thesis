\chapter*{Glossaire}

\begin{description}

\item[ActiveX]\mbox{}\linebreak 
Il s'agit d'un modèle de composants loiciels \emph{Componant Object Model} crée par Microsoft. Il permet le dialogue inter-programmes. \\ 
$\to$\url{http://fr.wikipedia.org/wiki/ActiveX}

\item[AJAX]\mbox{}\linebreak 
AJAX pour \emph{Asynchronous JavaScript And XML} est un acronyme désignant une méthode informatique de développement d'applications Web. Le terme AJAX s'applique lorsqu'un système Web utilise HTML, CSS, JavaScript, DOM et XMLHttpRequest \\ 
$\to$\url{http://fr.wikipedia.org/wiki/AJAX}

	\item[ARM]\mbox{}\linebreak 
C'est le nom donné à un type de processeur. ARM est l'acronyme de \emph{Advanced RISC Machines}. Ils sont fabriqué sur une architecture RISC de 32 bits. \\ 
$\to$\url{http://fr.wikipedia.org/wiki/Processeur_ARM}

	\item[BIPHASE]\mbox{}\linebreak 
Le code Biphase également appelé code de Manchester est un codage synchrone utilisé pour la transmission de données binaires sur un canal de communication. \\
$\to$\url{http://fr.wikipedia.org/wiki/BIPHASE_ou_Manchester}

	\item[BUS]\mbox{}\linebreak 
Un bus informatique sers à interconnecter différents matériels informatique. \\
$\to$\url{http://fr.wikipedia.org/wiki/Bus_informatique} 


\item[CEM]\mbox{}\linebreak 
La compatibilité électromagnétique est la capacité d'un appareil à résister à des ondes électromagnétiques ou l'aptitue d'un appereil à fonctionner dans un environnement soumis à des perturbations. \\
$\to$\url{http://fr.wikipedia.org/wiki/Compatibilit%C3%A9_%C3%A9lectromagn%C3%A9tique}   

\item[CLIENT]\mbox{}\linebreak 
Le terme de CLIENT s'applique à l'utilisation de LonTouch depuis un navigateur web executé sur un micro-ordinateur connecté à Ethernet. 

	\item[CSMA/CA]\mbox{}\linebreak   
CSMA/CA ou \emph{Carrier Sense Multiple Access with Collision Detect} est une méthode d'accès utilisé par des périphériques d'un bus informatique. \\
$\to$\url{http://fr.wikipedia.org/wiki/CSMA/CA} 

	\item[CSS]\mbox{}\linebreak 
Le langage CSS (\emph{Cascading Style Sheets} est utilisé pour décrire la présentation d'un document structuré écrit en HTML ou en XML. \\
$\to$\url{http://fr.wikipedia.org/wiki/CSS} 

	\item[DALI]\mbox{}\linebreak 
Le Protocole Dali ou Digital Adressable Lighting Interface est un protocole permettant le contrôle d'éclairages luminescent à ballastes. Il permet la gestion de 64 ballastes différents. \\
$\to$\url{http://fr.wikipedia.org/wiki/Protocole_Dali}

\item[DEBIAN]\mbox{}\linebreak   
Debian est un système d'exploitation libre pour un micro-ordinateur. Il s'agit d'une distribution sous licence GNU/Linux. \\
$\to$\url{http://fr.wikipedia.org/wiki/Debian}    

\item[DOXYGEN]\mbox{}\linebreak 
Doxygen est un logiciel informatique libre permettant de générer de la documentation à partie du code source d'un programme.\\
$\to$\url{http://fr.wikipedia.org/wiki/Doxygen} 

\item[DOM]\mbox{}\linebreak 
Le \emph{Document Object Model} abrégé DOM est une recommendation du W3C qui décrit une interface indépendante de tout langage de programmation et de toute plateforme. Il permet à des programmes informatiques notament des scripts du type JavaScript d'accéder ou de mettre à jour le contenu, la structure, ou le style de documents, par exemple, de HTML.  \\
$\to$\url{http://fr.wikipedia.org/wiki/Document_Object_Model}

	\item[FASTCGI]\mbox{}\linebreak 
		FastCGI est une méthode basée sur CGI \emph{Common Gateway Interface}. \\
$\to$\url{http://fr.wikipedia.org/wiki/Common_Gateway_Interface} 


\item[GECKO]\mbox{}\linebreak   
Gecko est un moteur d'affichage de pages Web utilisé par de nombreux navigateurs Web tels que Mozilla ou Firefox. \\
$\to$\url{http://fr.wikipedia.org/wiki/Gecko_%28moteur_de_rendu%29}

\item[GETTEXT]\mbox{}\linebreak 
Il s'agit d'un logiciel GNU maintenu par la \emph{Free Software Foundation} permettant de faciliter l'internationalisation d'un document. \\
$\to$\url{http://fr.wikipedia.org/w/index.php?title=GNU_Gettext&action=edit}

	\item[GET]\mbox{}\linebreak 
HTTP décrit une méthode permettant la réalisation de requêtes. Les trois méthodes possibles sont GET, POST et HEAD. \\
$\to$\url{http://www.themanualpage.org/http/http_http10.php}

	\item[GI]\mbox{}\linebreak 
Le terme GI est utilisé dans ce document pour faire référence aux Interface Graphiques accessible sur les TP et sur les navigateurs web des CLIENTS. GI est l'acronyme de \emph{Graphic Interface} 

	\item[GNU]\mbox{}\linebreak 
Le projet GNU à été lancé en 1984 au MIT afin de créer un système d'exploitation libre. GNU est un acronyme récursif signifiant \emph{GNU's Not UNIX}. \\
$\to$\url{http://fr.wikipedia.org/wiki/GNU}

	\item[GNU coding standards]\mbox{}\linebreak 
Convention de programmation érigées par la \emph{Free Software Foundation}. \\
$\to$\url{http://www.gnu.org/prep/standards/} 

	\item[Google]\mbox{}\linebreak 
Google est une société fondée en 1998. Elle est aujourd'hui connue par son moteur de recherche Internet. \\
$\to$\url{http://fr.wikipedia.org/wiki/Google} 

	\item[HTML]\mbox{}\linebreak 
L'\emph{Hypertext Markup Language}, généralement abrégé HTML, est un langage informatique utilisé pour écrire des pages Web. \\
$\to$\url{http://fr.wikipedia.org/wiki/HTML} 

	\item[HTTP]\mbox{}\linebreak 
L'\emph{Hypertext transfert Protocol} abrégé HTTP utilisé principalement pour transférer des documents HTML entre un serveur HTTP et un navigateur Web. \\
$\to$\url{http://fr.wikipedia.org/wiki/HTML} 

	\item[IP]\mbox{}\linebreak 
IP est un protocole de la couche 3 du modèle OSI (couche réseau). Il gère le transport et le routage des paquets sur un réseau informatique. \\
$\to$\url{http://fr.wikipedia.org/wiki/Protocole_Internet}

	\item[JavaScript]\mbox{}\linebreak 
Il s'agit d'un langage de programmation connu sous la dénomination d'ECMAScript. Il est défini par la norme ECMA-262 et est utilisé couramment dans des pages Web. \\
$\to$\url{http://fr.wikipedia.org/wiki/JavaScript}

	\item[IBT]\mbox{}\linebreak 
IBT est le nom de la société partenaire encadrant ce travail de diplôme. IBT pour \emph{Ingenieurbüro Brönnimann Thun}. \\
$\to$\url{http://www.ibt.ch}


\item[LATEX]\mbox{}\linebreak 
\LaTeX~ est une collection de macto-commandes pour \TeX crée par l'illustre \emph{Donald Ervin Knuth} destiné à faciliter l'utilisation de ce processeur de texte. Le présent document est réalisé à l'aide de \LaTeX.\\
$\to$\url{http://fr.wikipedia.org/wiki/LaTeX} 

\item[LDI]\mbox{}\linebreak   
Est l'abréviation de \emph{LC709 Driver Interface}. Il s'agit de l'interface entre la carte d'accès à LonWorks et la librairie Orion.

\item[LINUX]\mbox{}\linebreak 
Linux désigne un système d'exploitation de type UNIX développé par \emph{Linux Torvalds} en 1991. \\
$\to$\url{http://fr.wikipedia.org/wiki/Linux} 

	\item[LON]\mbox{}\linebreak 
LonWorks abrégé LON est une technologie de bus de terrain utilisée dans l'automatisation du contrôle de bâtiments. 
$\to$\url{http://en.wikipedia.org/wiki/LonWorks} (version anglaise)

	\item[LonTalk]\mbox{}\linebreak 
LonTalk est le nom du protocole de communication utilisé dans la technologie LonWorks. 
$\to$\url{http://en.wikipedia.org/wiki/LonTalk} (version anglaise)

	\item[LonWorks]\mbox{}\linebreak 
LonWorks abrégé LON est une technologie de bus de terrain utilisée dans l'automatisation du contrôle de bâtiments. 
$\to$\url{http://en.wikipedia.org/wiki/LonWorks} (version anglaise)

	\item[LTCH]\mbox{}\linebreak 
Désigne une abréviation utilisée dans ce document pour parler du projet LonTouch.  


\item[L-CORE]\mbox{}\linebreak 
Le L-Core, développé par la société Loytec est un controller hardware basé sur le circuit LC3020 et le système d'exploitation RTEMS. \\
$\to$\url{http://www.loytec.com/english/products/lcore.htm}  

\item[MAKE]\mbox{}\linebreak 
make est un logiciel traditionnel d'UNIX. Il sert à appeler des commandes créant des fichiers. \\
$\to$\url{http://fr.wikipedia.org/wiki/Make}  

\item[Manchester]\mbox{}\linebreak 
Le code de Manchester également appelé code Biphase est un codage synchrone utilisé pour la transmission de données binaires sur un canal de communication. \\
$\to$\url{http://fr.wikipedia.org/wiki/BIPHASE_ou_Manchester} 		

\item[NEURON ID]\mbox{}\linebreak 
Les périphériques LonWorks sont identifiés par un numéro unique codé sur $48[bits]$ appelé \emph{Neuron Id} 

\item[NL220]\mbox{}\linebreak   
NL220 est un logiciel de configuration d'un réseau LonWorks créé par la société française Newron Systems.  \\
$\to$\url{http://www.newron-system.com/fr/nl220TE.php}

\item[NV]\mbox{}\linebreak   
Est l'abréviation de \emph{Network Variable}.

	\item[OBJET]\mbox{}\linebreak 
Le terme OBJET s'applique ici à la programmation orientée objet abrégée POO. Il s'agit d'une façon d'architecturer une application informatique en regroupant les données en entités, les objets. \\
$\to$\url{http://fr.wikipedia.org/wiki/Programmation_objet} 

	\item[OCTET]\mbox{}\linebreak   
Un octet est une unité de mesure informatique. Un octet est composé de $8[bits]$. \\
$\to$\url{http://fr.wikipedia.org/wiki/Octet}


\item[ORION]\mbox{}\linebreak 
L'Orion Protocole Stack est une librairie logicielle issue de la technologie Loytec. Développée en langage C, elle permet d'interaction avec un réseau LonWorks\\
$\to$\url{http://www.loytec.com/english/products/orion.htm}   

\item[OSI]\mbox{}\linebreak   
Le modèle OSI (\emph{Open Systems Interconnection}) permet de décrire en 7 couches d'abstraction le fonctionnement d'un protocole de communication informatique. \\
$\to$\url{http://fr.wikipedia.org/wiki/Octet} 

\item[OSSI]\mbox{}\linebreak   
Ossi est une librairie logicielle créée par la société Loytec pour rendre portable la librairie Orion.

	\item[OVERHEAD]\mbox{}\linebreak   
L'\emph{overhead} est un terme informatique définissant la surcharge temporelle dù à l'utilisation d'un protocole de communication ou d'un programme informatique. 


\item[PC]\mbox{}\linebreak   
Il s'agit d'un terme utilisé pour désigner un ordinateur personnel encore appelé micro-ordinateur ou ordinateur indivituel. Le terme PC est en réalité une marque déposée d'IBM. \\
$\to$\url{http://fr.wikipedia.org/wiki/Ordinateur_personnel}

\item[PDF]\mbox{}\linebreak   
Le \emph{Portable Document Format} abrégé PDF est un format de fichier qui préserve les polices, les images, les objets graphique et la mise en forme du document source. Ce format à été crée par la Adobe Systems. \\
$\to$\url{http://fr.wikipedia.org/wiki/Portable\_Document\_Format}

\item[PORT]\mbox{}\linebreak 
Correspondant à la couche de session du modèle OSI, la notion de port permet de distinguer différents interlocuteurs. \\
$\to$\url{http://www.iana.org/assignments/port-numbers} \\
$\to$\url{http://fr.wikipedia.org/wiki/Port_logiciel}

\item[POST]\mbox{}\linebreak
POST est l'une des trois méthodes possible pour constituer une requête HTTP. 

	\item[RAM]\mbox{}\linebreak   
La mémoire vive, aussi appelée RAM pour \emph{Random Access Memory} fait référence à un type de mémoire volatile mais ultra rapide utilisée principalement dans l'informatique. 
$\to$\url{http://fr.wikipedia.org/wiki/RAM} 

\item[RTEMS]\mbox{}\linebreak   
RTEMS pour \emph{Real-Time Operating System for Multiprocessor Systems} est un système d'exploitation utilisé par la librairie OSSI de Loytec dans le cas d'un hardware embarqué. \\
$\to$\url{http://www.rtems.com/}  

	\item[SERVEUR]\mbox{}\linebreak   
SERVEUR est un terme employé dans ce document pour désigner la machine maîtresse du système LonTouch. \\

	\item[TELNET]\mbox{}\linebreak   
Telnet est un protocole réseau basé sur TCP. Il fait parti des protocoles de la couche 7 du modèle OSI. \\
$\to$\url{http://fr.wikipedia.org/wiki/Telnet}

	\item[TP]\mbox{}\linebreak   
Les \emph{TouchPanels} ou écrans tactiles sont les ordinateurs destinés à être imbriqué dans les murs d'un bâtiment et relié au SERVEUR LonTouch par l'intermédiaire d'un canal de communication. 


\item[UNIX]\mbox{}\linebreak   
UNIX est le nom d'un système d'exploitation créé en 1969 pour un usage principalement professionnel. Le système Linux est fortement inspiré du fonctionnement d'UNIX. \\
$\to$\url{http://fr.wikipedia.org/wiki/Unix}

\item[URL]\mbox{}\linebreak   
L'\emph{Uniform Resource Locator} désigne une adresse réticulaire. Il s'agit vulgairement parlant d'une adresse Internet. \\
$\to$\url{http://fr.wikipedia.org/wiki/URL}
	\item[W3C]\mbox{}\linebreak   
Le \emph{World Wide Web Consortium} abrégé W3C, est un consortium fondé en octobre 1994 pour promouvoir la compatibilité des technologies du \emph{World Wide Web} telle que HTML, XHTML ou CSS. \\
$\to$\url{http://fr.wikipedia.org/wiki/W3c}

	\item[WEB]\mbox{}\linebreak 
Le \emph{World Wide Web} communément appelé Web ou parfois Toile est un système hypertexte public fonctionnant principalement sur Internet permettant de consulter des pages Web mises en ligne dans des sites Web. \\
$\to$\url{http://fr.wikipedia.org/wiki/Web} 


\item[WINDOWS]\mbox{}\linebreak   
Windows est une gramme de système d'exploitation produite par la société Microsoft. Ces systèmes sont principalement prévu pour tourner sur des PC \\
$\to$\url{http://fr.wikipedia.org/wiki/Microsoft\_Windows}

\item[XML]\mbox{}\linebreak   
L'\emph{Extensible Markup Language} est un standard du w3c qui sers de base pour créer des langage de balisage spécialisés. \\
$\to$\url{http://fr.wikipedia.org/wiki/Web}

\item[ZIP]\mbox{}\linebreak   
ZIP est un format de compression de fichier utilisé en informatique. \\
$\to$\url{http://fr.wikipedia.org/wiki/Zip#Format_de_compression} 

\end{description}
